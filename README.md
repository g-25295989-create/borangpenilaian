# MARCHING PRO v6 - MECHA WHITE

Sistem penilaian kawad kaki moden dengan estetik robotik "Mecha White". Dibangunkan khusus untuk pengurusan pemarkahan rumah sukan dengan integrasi Google Sheets.

## Ciri-ciri Utama
- **Antaramuka Mecha**: Reka bentuk bersih dengan aksen Merah dan Gold.
- **Sistem Penilaian**: 5 kriteria utama (Kekemasan, Kawalan, Kepimpinan, Disiplin, Keseluruhan).
- **Analisis Live**: Paparan graf bar prestasi unit secara masa nyata.
- **Integrasi Google Sheets**: Penghantaran data secara automatik ke pangkalan data awan.

## Teknologi
- React 19
- Tailwind CSS
- Recharts (Analisis Visual)
- Google Apps Script (Backend)

## Pemasangan
1. Clone repositori ini.
2. Jalankan `npm install`.
3. Jalankan `npm run dev` untuk pembangunan tempatan.
4. Jalankan `npm run build` untuk pengeluaran.
